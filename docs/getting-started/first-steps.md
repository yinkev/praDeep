# First Steps

This guide walks you through your first meaningful interactions with DeepTutor.

## Understanding the Interface

### Main Components

| Component | Purpose |
|-----------|---------|
| **Document Panel** | View and navigate uploaded documents |
| **Chat Interface** | Interact with the AI assistant |
| **Mode Selector** | Switch between different interaction modes |
| **Citation Panel** | View sources and references |

## Interaction Modes

### 1. Smart Q&A Mode

The default mode for document-based questions and answers.

**Best for:**
- Understanding complex concepts
- Getting explanations with citations
- Clarifying specific sections

### 2. Practice Mode

Generate custom quizzes and practice problems.

**Best for:**
- Exam preparation
- Self-assessment
- Active recall practice

### 3. Deep Research Mode

Systematic topic exploration with web integration.

**Best for:**
- Literature reviews
- Topic deep-dives
- Connecting concepts across sources

### 4. Co-Writer Mode

AI-assisted writing and editing.

**Best for:**
- Drafting documents
- Editing and refinement
- Adding citations automatically

## Tips for Effective Use

1. **Be Specific**: The more specific your question, the more precise the answer
2. **Use Context**: Reference specific sections or pages when relevant
3. **Iterate**: Follow up on answers to go deeper into topics
4. **Verify**: Always check citations for critical information

## Common First Tasks

- [ ] Upload a study document
- [ ] Ask a conceptual question
- [ ] Generate practice questions
- [ ] Try the research mode on a topic
- [ ] Use co-writer to draft a summary
